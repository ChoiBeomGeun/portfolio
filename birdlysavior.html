Birdy Savior

    Unity Engine을 이용한 개인 모바일 게임 프로젝트 입니다. 간단하지만 중독성있는 3D 러닝게임 입니다.


  • 개발 시기 : 2018년 10월 ~ 2018년 11월
    게임 장르 : 3D 아케이드 러닝 게임

  • 사용 엔진 : Unity Engine
  • 사용 언어 : C#
  • 플랫폼 : Android
  • 마일스톤 : 파이널
  • 제작 : 최범근 1인 개발 작품

    • 1. 사이드 뷰 러닝게임 구현

    랜덤으로 생성되는 나무들을 피하는 사이브 뷰 러닝 게임 구현.

    2. 탑 뷰 게임 구현

    랜덤으로 생성되는 유성들을 피하는 탑 뷰 게임 구현.

    3.1인칭 게임 구현
    1인칭을 진행되는 게임 구현.
    적들로 부터 지구를 지켜내야함.

    4.레벨 업 시스템

    특정 점수 구간을 지나면 게임의 난이도가 증가함
    장애물 스폰 간격이 줄어들고 장애물들의 속도가 증가함

    5.상점 구현

    구글 플레이 스토어를 통한 인앱 결제 구현 및 게임 속 코인으로 구매 가능한 물품들이 존재함.

    6.랭킹 및 업적 시스템


    구글 플레이를 통한 업적 및 랭킹 시스템

    6.Pause UI

    게임 퍼즈 기능을 구현함
    다시 시작,레벨 선택으로 돌아가기,게임으로 돌아가기, 음량 조절이 가능함
