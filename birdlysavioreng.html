Birdy Savior

    This is a personal mobile game project using Unity Engine. This game is very simple but you'll be addicted


  • Project Duration: 2018.10 ~ 2018.11
    Game Genre : 3D Arcade Runner Game

  • Used Engine : Unity Engine
  • Used Language : C#
  • Platform : Android
  • Milestone : Final
  • Made by BeomGeun Choi

    • 1.Side View Running Game

    Side view Running Game which player must dodge trees are generated randomly

    2.Top View Game

    Top view Game which player must dodge meteors are generated randomly

    3.FPS Game

    This game is First-Person Shooter game
    Player should save the earth from enemies

    4.Level Up System

    Difficulty of the game will be changed as the player's score.
    Obstacles will be spawned speedily and obstacles will move speedily

    5.Shop

    Using Google Play, In-App Purchase is available.
    6.Rank and Achievements


    Using Google Play, there are rank and achievements system

    7.Pause UI

    Pausing Game is supported
    There are "Restart","Go to Level Select", "Return", "Adjust Volume"
