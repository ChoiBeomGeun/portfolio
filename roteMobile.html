RoteRote Mobile


    Unity Engine을 이용한 개인 모바일 게임 프로젝트 입니다.2.5년간 PC게임만 개발하던 저는 모바일 게임 제작에 임해보고 싶었습니다. 그래서 가장 애정이 깊던 프로젝트인 RoteRote 게임을 모바일 버전으로 개발하였습니다. PC RoteRote의 게임 디자인을 Unity Engine을 이용하여 구현하였습니다.


  • 개발 시기 : 2018년 6월 ~ 2018년 8월
    게임 장르 : 2D 플랫포머 퍼즐 게임

  • 사용 엔진 : Unity Engine
  • 사용 언어 : C#
  • 플랫폼 : Android
  • 마일스톤 : 베타
  • 제작 : 최범근 1인 개발 작품

    • 1.Json 파일을 이용한 레벨 생성

    LitJson을 사용해서 기존 PC버전의 RoteRote 레벨을 유니티 엔진에서 로딩함.
    오브젝트 스타일에 따라 지정된 프리팹을 생성함.

    2.월드 회전 트리거

    플레이어와 충돌하면 지정된 각도로 화면을 회전시키는 트리거

    3.게임 로직 구현
    클리어 존에 도달하면 다음 레벨로 넘어가는 로직.
    플레이어와 떨어지는 박스 사이의 겹치는 충돌 점이 3개 이상이 존재하면 플레이어가 박스에 깔렸다고 가정하여 게임을 다시 시작하게 함.

    4.플레이어 컨트롤러

    좌우 이동, 벽 점프, 누르는 정도에 따른 점프 세기등을 구현함
    카메라의 각도가 0도, 90도, 180도, 270도로 바뀔 때마다 플레이어 컨트롤러가 변화됨.
    다양한 프로퍼티를 조정할 수 있어서 플레이어 컨트롤러의 세밀한 조정이 가능함

    5. UI 구현
    게임내에서 다시 시작, 레벨 선택으로 돌아갈 수 있는 UI를 구현함

    6. 메인 메뉴, 레벨 선택, 옵션 스테이트 구현

    메인 메뉴에서 레벨 선택, 옵션 스테이트로 이동 ,게임 종료가 가능함
    레벨 선택에서 지정한 레벨을 실행하거나 다시 메뉴 스테이트로 이동이 가능함.
    옵션 스테이트에서 게임 소리를 끄거나 켤 수 있고 크레딧을 볼 수 있음.
